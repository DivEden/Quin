Problem: "python is not recognized"
Solution: Python not in PATH. Reinstall Python with "Add to PATH" checked.

Problem: "No module named 'selenium'"
Solution: Run: pip install selenium

Problem: Browser doesn't open
Solution: Make sure Chrome is installed. Selenium 4.6+ auto-downloads ChromeDriver.

Problem: "Worker file not found"
Solution: Run the script from the quinix-workers folder where all worker files are.

Problem: Workers don't start
Solution: Make sure you're logged into Quinyx. Check browser console for errors.
