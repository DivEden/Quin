Q: Console window doesn't appear
A: Make sure tkinter is installed (comes with Python by default)

Q: Window appears but stays blank
A: Check the Python terminal window for errors

Q: Can't see some colors
A: Colors are optimized for dark theme. Window uses #1e1e1e background.

Q: Window is too small/large
A: Drag the edges to resize, or edit quinix_tab_manager.py line ~56:
   self.root.geometry("1000x600")  â† Change these numbers

Q: Window disappears behind browser
A: The window should be always-on-top. If not, there might be
   a conflict with other always-on-top software.

Q: Too many messages, hard to follow
A: Console keeps last 1000 messages. Older ones auto-delete.
   You can adjust STATUS_CHECK_INTERVAL in the script to reduce
   message frequency.

Q: Want to copy text from console
A: Click and drag to select text, then Ctrl+C to copy
